# Candy-Detection-system
,Developed a YOLO-based object detection model trained to identify and classify multiple candy products in real time using computer vision techniques.The model is trained to detect the following candy types:  MMs_peanut ,MMs_regular, airheads ,gummy_worms, milky_way, nerds, skittles, snickers, starbust, three_musketeers, twizzlers.
